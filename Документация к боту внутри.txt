========= файлы бота ==========

rapbot.py - сердце бота, кароче фулл код бота

admin_panel.py - код админ панели (ее сердце)
admin_panel.bat - запуск админ панели
start_bot.bat - резерв для админ панели, если какая-то сдохнет

rapper.json - карточки всех реперов
promo.json - промокоды

backup - папка с резервами баз данных
bjd.db - база данных на данный момент


========= Команды бота в Tg ========

/start - запустить бота
/help - помощь по боту
/suppot - поддержка бота
/rapper - выбить карточку репера
/coll и /collection - посмотреть своб коллекцию
/profile - посмотреть свой профиль
/promo - ввести промокод
